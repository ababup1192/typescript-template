# Typescript Template

[![CircleCI](https://circleci.com/gh/ababup1192/typescript-template.svg?style=svg)](https://circleci.com/gh/ababup1192/typescript-template)

# Preparation

```
$ brew install yarn

$ yarn
$ yarn run typings install
```

# Testing
```
$ yarn test
```

# Build
```
$ yarn build
```
